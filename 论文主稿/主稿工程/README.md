# 主稿工程目录索引

## 核心文件
- `LF-CBO_Manuscript.tex`: 当前主稿源文件。
- `LF-CBO_Manuscript.pdf`: 当前编译后的主稿 PDF。
- `LF-CBO_References.bib`: 参考文献数据库。
- `sn-jnl.cls`: Springer Nature 模板类文件。

## 核心子目录
- `figures/`: 主稿使用的图。
- `data/`: 修订分析的汇总数据。
- `scripts/`: 当前仍在使用的生成与打包脚本。
- `bst/`: 参考文献样式文件。

## 辅助子目录
- `管理文档/`: 已按用途拆分为选刊与投稿策略、交付复核、数据来源说明、内部说明和投稿模板。
- `工具归档/`: 早期一次性脚本和临时输出，默认不参与当前主流程。

## 在工作区中的位置
- 上游修订数据来自 `../04_修订数据/`。
- CloudSim 补充材料位于 `../05_CloudSim补充包/`。
- 参考文献核查位于 `../07_参考文献核查/`。
- 生成的源投稿包输出到 `../02_投稿源包/`。
- 可直接投递的整理成品位于 `../03_待发布/`。

## 当前主流程
- 编译主稿：`LF-CBO_Manuscript.tex`
- 生成投稿包：`scripts/create_journal_submission_packages.py`

## 管理文档分区
- `管理文档/01_选刊与投稿策略/`: 目标期刊筛选与投稿顺序建议。
- `管理文档/03_交付与复核/`: 投稿前差距清单和全稿复核结果。
- `管理文档/04_数据与来源说明/`: 数据来源、证据链和补充说明。
- `管理文档/90_内部说明/`: 面向本地工作流的内部说明，不属于投稿材料。
- `管理文档/91_投稿模板/`: 历史保留的投稿模板与检查清单。
